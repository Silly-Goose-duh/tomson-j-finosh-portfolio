# Tomson J. Finosh | Portfolio

A modern, clean portfolio website showcasing my work, skills, and projects.

## Live Demo

🔗 **[tomson-j-finosh.vercel.app](https://tomson-j-finosh.vercel.app)**

## Tech Stack

- **React** - UI Framework
- **Create React App** - Project Setup
- **Vercel** - Deployment

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Project Structure

```
portfolio/
├── public/           # Static assets
├── src/             # React components
├── index.html       # Entry point
└── package.json    # Dependencies
```
